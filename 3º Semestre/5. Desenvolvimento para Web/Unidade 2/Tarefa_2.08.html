Exemplo de Lista de Descrição

TCP
O Protoclo de Controle de Transmissão (TCP) é um método (protocolo) usado junto com o Protocolo de Internet (IP) para enviar dados na forma de unidades de mensagens, chamadas de pacotes, entre computadores pela internet.
IP
O Protocolo de Internet (IP) é o método ou protocolo pelo qual os dados são enviados de um computador para outro na internet. Cada computador na internet é identificado de forma única por um endereço IP.
FTP
O Protocolo de Transferência de Arquivos (FTP) é um protocolo usado para trocar Arquivos entre computadores na internet.
HTTP
O Protocolo de TRansferência de Hipertexto (HTTP) é um protoclo usado para trocar texto, imagens gráficas, som, vídeo e outros arquivos multimídia na Web.
