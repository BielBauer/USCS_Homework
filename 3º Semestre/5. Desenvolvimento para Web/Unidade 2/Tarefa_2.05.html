A Linguagem de Programação é uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas e softwares.

Reforçando esse conceito, Silas de Carneiro define:

"Entende-se por linguagem de programação ao conjunto de regras sintática definidas pelo usuário e que devem ser executadas pelo computador para a realização de tarefas, por exemplo, a solução numérica de problemas de problemas matemáticos ou aquisição de medidas de experimentos em laboratório."
