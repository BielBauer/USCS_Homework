Questões:

1. Descrever a origem, propósito e características da linguagem HTML.

Origem: HTML (Hypertext Markup Language) foi desenvolvido por Tim Berners-Lee em 1990, enquanto trabalhava no CERN. Foi a base da World Wide Web, permitindo a criação de documentos que contêm links para outros documentos, formando a estrutura básica da web.
Propósito: O propósito principal do HTML é estruturar e apresentar informações em formato de hipertexto, permitindo a criação de documentos interconectados (hipertexto) acessíveis através de navegadores da web.
Características: HTML é uma linguagem de marcação que utiliza tags para estruturar e formatar o conteúdo. As características incluem:
Utilização de tags: As tags são elementos marcados entre colchetes angulares ("<" e ">") que indicam ao navegador como exibir o conteúdo.
Estrutura hierárquica: As tags são organizadas hierarquicamente para criar a estrutura do documento, permitindo aninhar elementos dentro de outros elementos.
Hipertexto: Links (âncoras) podem ser criados para conectar diferentes documentos ou partes do mesmo documento.
Suporte a multimídia: Elementos como imagens, vídeos e áudio podem ser incorporados nas páginas.
Formulários interativos: Elementos de formulário permitem a coleta de informações dos usuários.
Semântica: Tags possuem significado semântico, o que ajuda os motores de busca e leitores de tela a compreenderem o conteúdo.

2. Descreva o software necessário para se criar e testar páginas HTML.

Editor de Texto: Um simples editor de texto (como Bloco de Notas no Windows ou TextEdit no macOS) pode ser usado para criar páginas HTML, embora não ofereçam recursos avançados de edição.
IDE (Integrated Development Environment): IDEs como Visual Studio Code, Sublime Text ou Atom oferecem recursos avançados, como realce de sintaxe, sugestões de código e extensões para facilitar a criação de páginas HTML.
Ferramentas de Design Web: Softwares como Adobe Dreamweaver ou Pinegrow permitem criar páginas HTML visualmente, sem necessidade de escrever código.
Navegador Web: Qualquer navegador moderno (como Chrome, Firefox, Edge) pode ser usado para testar páginas HTML, basta abrir o arquivo localmente no navegador.

3. Descreva a finalidade das seções "head" e "body" de uma página HTML.

"head": Esta seção contém metadados e informações sobre o documento. Ela não é exibida diretamente na página para o usuário, mas fornece informações importantes para o navegador e motores de busca. Exemplos de elementos comuns na seção incluem:
"title": Define o título da página exibido na barra de título do navegador.
"meta": Fornece metadados sobre a página, como codificação de caracteres e descrição.
"link": Conecta a página a arquivos externos, como folhas de estilo.
"script": Inclui scripts JavaScript, geralmente referenciando arquivos externos.
"body": Esta seção contém o conteúdo visível da página, que é exibido para os usuários. Elementos como texto, imagens, links, vídeos e outros elementos de mídia são colocados dentro desta seção. É onde a maior parte da estrutura e do conteúdo da página é definida.
